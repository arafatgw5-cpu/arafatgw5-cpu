## 💫 About Me

I am **Easin Arafat**, an aspiring **Full Stack / Frontend Web Developer** who loves building clean and useful web applications.

My main focus is creating **modern UI**, writing **clean code**, and building projects with **React, Next.js, Tailwind CSS, Node.js, Express.js, MongoDB, and authentication systems**.

I believe every project is a chance to learn something new and become a better developer.

### 🔥 What I’m Working On

- 🚀 Building real-world projects with **Next.js App Router**
- 🎨 Improving UI/UX with **Tailwind CSS** and **DaisyUI**
- 🔐 Learning **Better Auth**, Google OAuth, and GitHub OAuth
- 🗄️ Practicing backend with **Express.js** and **MongoDB**
- 🌐 Working with REST APIs and dynamic routes
- 📚 Improving debugging, clean code, and problem-solving skills
